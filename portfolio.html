AG4NBA — Automated Luggage Handling

An optimization algorithm that automates luggage packing at Paderborn-Lippstadt Airport, maximising container utilisation and driving an end-to-end system: sensor data management, luggage tracking, placement algorithms and robotic-arm integration, on a Python microservice backend with PostgreSQL.

↓ ~20% fewer bins per flight · ~30 hrs/week manual work saved

FastGate — Aircraft Localisation

A system that detects and tracks an aircraft and automates docking of the boarding bridge without human involvement, built as a set of containerised microservices with point-cloud alignment modules using Open3D and CAD-based localisation.

◎ 5–10 cm localisation accuracy · autonomous docking

An OpenCV system that detects eye blinks through grayscale image processing for assistive communication, with face/eye detection and blink-recognition logic — the basis for a peer-reviewed publication on encrypted communication using Morse code.
